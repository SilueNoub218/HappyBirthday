# Happy Birthday App

## Description

Happy Birthday App est une application Android développée avec **Kotlin** et **Jetpack Compose**. Elle affiche une carte d'anniversaire composée d'une image d'arrière-plan et d'un message de vœux centré sur l'écran.

## Fonctionnalités

* Affichage d'un message d'anniversaire.
* Affichage du nom de l'expéditeur.
* Image de fond avec transparence.
* Interface développée avec Jetpack Compose.
* Utilisation du thème Material 3.

## Technologies utilisées

* Kotlin
* Android Studio
* Jetpack Compose
* Material 3
